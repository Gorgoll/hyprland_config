## to install everything you'll need (probably...):
```bash 
sudo pacman -Syu
sudo pacman -S --needed swww waybar base-devel git
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
yay -S wlogout xcursor
```
## just put this in: 
```bash 
~/.config
```
## the icons folder is supposed to go to: 
```bash 
/usr/share/wlogout/icons
```
## put the wlogout folder to 
```bash
/etc
```
## To top it up with the wallpaper:
```bash 
here's the link to the wallpaper:
https://drive.google.com/file/d/1QTY6zVAIGCwfMnP1MZnSOSJM36pbwjGR/view?usp=sharing
swww img ~/path
```
